Required Dependencies

BigBrain is required - FriendlyPMC is built on SAIN 4.2.0, which requires BigBrain:

  • BigBrain - Required AI decision framework

Clean installation required: Remove old server mod files and the old SAIN directory before installing.

2

Install Mod

Copy all FriendlyPMC files and folders into your SPT root folder. No additional server mod configuration needed.

3

Configure Features

Launch SPT and press F6 in-game to access FriendlyPMC settings:

  • Companion Count: Set 1-4 companions (in UI)
  • Enable Medic Role: OFF by default (alpha feature)
  • Enable Medic Evac: OFF by default (experimental)
  • Gestures & Phrases: Enabled by default

‚ùì Frequently Asked Questions

Q: What dependencies do I need?

A: BigBrain is required. FriendlyPMC is built on SAIN 4.2.0, which requires BigBrain. Download from SPT Forge.

Q: What's changed in Playtest 5?

A: This release removes the need for separate server mods and simplifies spawning. BigBrain is the only external dependency required. Companion spawning is now client-side (Spawn V2).

Q: How many companions can I have?

A: 1-4 companions, configurable directly in the F6 UI. No server-side configuration required.

Q: What is the Medic Evac feature?

A: Extremely experimental. When critically wounded, your squad stabilizes you and carries you to the evac zone. Enable in config to test. We want to hear your experience!

Q: My companions are stuck looting, what do I do?

A: Known bug being fixed. Try using gesture commands or running away to move them along. Combat will force them into a new state and break the deadlock.

Q: Is this compatible with other mods?

A: Compatibility with ABPS (Acid's Bot Placement System) is unknown - we made efforts to ensure compatibility, but need playtesters to confirm. If you're testing with ABPS, please report your results!

Q: How do I control my companions?

A: Use the F1 gesture wheel. Native phrases and gestures are enabled by default and integrate with the AO (Area of Operations) system.

Q: Is healing still free?

A: Yes! Companion healing is free until Medic V2 is released. The alpha Medic Role can be enabled in config for additional features.

Q: Can I download this from the SPT Forge?

A: No. Unfortunately, before the mod could even really get started, the mods accused the developer of lying about the planned features (most of which are now delivered as promised) and banned the dev from SPT Forge and SPT Discord for being a "vibe coding AI snake oil salesman".

The result: FriendlyPMC must be distributed outside of SPT Forge. Download links and installation instructions are provided above.
